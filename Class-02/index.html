Class-02

In this class we cover following lessons:

  1. Math Expressions with values (In this we directly calculate our values in our code)
  2. Familiar Looking Math Expressions (This lesson teaches you how to write math in JavaScript the way you write it in school.)
  3. Round of values for better reading (Used .toFixed for rounding values from 342.5678 to 342)
  4. Alerting Variable Values (In this we calculate values and display them in our browser using alert like [alert("Name: " + name + " | Age: " + age);)]
  5. Line Breaking in JS (We learn how to brak line in JS)
